# My dotfiles

- 通过`GNU STOW`来进行软链接
- 通过主机名判断来兼容多台设备
- 目前配置文件有：

  1. zsh
  2. tmux
  3. helix
  4. neofetch
  5. yabai
  6. skhd
  7. sketchybar
  8. clang-format
  9. p10k
  10. wezterm
  11. lazygit
  12. ideavim
  13. polybar
  14. conky
  15. vscode

- `stow.sh`脚本可以一键更新链接
- `save_vscode_extensions.sh`脚本作为**pre-commit**钩子，可以自动保存vscode插件列表
