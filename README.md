# My dotfiles

- 通过`GNU STOW`来进行软链接
- 通过主机名判断来兼容多台设备
- 目前配置文件有：

  1. zsh
  2. tmux
  3. helix
  4. neofetch
  5. yabai
  6. skhd
  7. sketchybar

- `stow.sh`脚本可以一键更新链接
